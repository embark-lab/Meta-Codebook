Overview

This codebook documents variables from three studies — POWERED, MAXED, and MR21 — grouped into six conceptual categories, with a seventh Administrative / Other appendix for REDCap system fields (timestamps, record IDs, form completion flags, consent/signature fields, etc.). Each variable is tagged with the project(s) in which it appears:

htmltools::HTML(project_badges_vec("POWERED,MAXED,MR21"))
POWERED MAXED MR21

Each chapter below has one searchable, sortable, filterable table. Use the column filters at the top of each table to narrow by Measure or Project.

codebook |>
  dplyr::count(category, name = "Variables") |>
  dplyr::arrange(dplyr::desc(Variables)) |>
  dplyr::rename(Category = category) |>
  DT::datatable(options = list(dom = "t", pageLength = 10), rownames = FALSE)
codebook |>
  tidyr::separate_rows(projects, sep = ",") |>
  dplyr::mutate(projects = trimws(projects)) |>
  dplyr::filter(projects != "") |>
  dplyr::count(category, projects) |>
  tidyr::pivot_wider(names_from = projects, values_from = n, values_fill = 0) |>
  dplyr::rename(Category = category) |>
  DT::datatable(options = list(dom = "t", pageLength = 10), rownames = FALSE)
